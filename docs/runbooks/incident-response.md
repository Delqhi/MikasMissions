# Incident Response Runbook

## Trigger
Any SLO breach, outage, or child-safety policy failure.

## Process
1. Classify severity (SEV-1 to SEV-3).
2. Assign incident commander and communication lead.
3. Stabilize user impact first.
4. Capture timeline and evidence.
5. Produce postmortem within 48 hours.

## Exit Criteria
- Service restored
- Mitigation validated
- Follow-up actions tracked with owners and deadlines
